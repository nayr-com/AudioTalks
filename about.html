The Project

This project was started in order to give other people quick access to speech's from conferences and other events. I really enjoy listening to talks when walking to and from classes so I wanted to make a website where I (and others) could quickly find a talk to listen to via download or stream. This helps to circumvent the arduous process of finding a talk and converting it yourself.

If you have suggestions for talks please send them to my e-mail: linked here. The site is hosted through GitHub and the source code is located here.
